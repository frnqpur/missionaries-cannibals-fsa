# Portfolio Page Template

## Purpose

This folder contains a static landing page / case study page for:

**FSA-Based Missionaries and Cannibals Puzzle Game**

The page explains the project to recruiters. It is not a web game and does not recreate the Pygame game in HTML/CSS/JavaScript.

The focus is:

- original Python/Pygame implementation,
- Windows executable build,
- screenshots,
- video demo,
- GitHub link,
- local setup instructions,
- recruiter-friendly project explanation.
