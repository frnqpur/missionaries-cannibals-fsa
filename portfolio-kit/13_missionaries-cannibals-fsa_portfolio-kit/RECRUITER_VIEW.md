# Recruiter View: FSA-Based Missionaries and Cannibals Puzzle Game

## 30-second recruiter summary

**FSA-Based Missionaries and Cannibals Puzzle Game** is a Python/Pygame desktop puzzle game that visualizes finite state automata concepts through the classic Missionaries and Cannibals problem. The project demonstrates state representation, rule validation, event-driven game programming, asset handling, and Windows executable packaging with PyInstaller.

The project is currently positioned as an **original Pygame runnable application plus Windows executable**, supported by a demo video, downloadable release, GitHub repository, and portfolio page. It is not primarily presented as a web game.

## What this project demonstrates

- Ability to convert a formal logic problem into an interactive application.
- Understanding of finite state automata and state transitions.
- Practical Python application development using Pygame.
- Event-driven input handling with mouse clicks and game loop updates.
- Rule validation for safe and unsafe puzzle states.
- Packaging a Python desktop app into a Windows executable for non-technical users.
- Portfolio-ready project documentation and deployment planning.

## Suggested interview talking points

1. **State representation**
   - Explain why the game state is represented as `[M_left, C_left, Boat]`.
   - Describe how boat direction affects the next state.

2. **Finite State Automata concept**
   - Explain how each valid move is a transition from one state to another.
   - Explain how invalid states trigger game-over conditions.

3. **Rule validation**
   - Describe the safety rule: missionaries cannot be outnumbered by cannibals on either bank when missionaries are present.

4. **Pygame implementation**
   - Explain the event loop, mouse click handling, character selection, and animation flow.

5. **Executable packaging**
   - Explain why PyInstaller folder mode was used and how image/music assets are bundled.

6. **Portfolio readiness**
   - Explain how the project is prepared for GitHub, demo video, screenshots, Windows download, and portfolio deployment.
